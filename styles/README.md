# Styles

Los estilos globales viven en `/app/globals.css` porque Next.js App Router importa CSS global desde el layout principal.

Usar Tailwind CSS para layout, spacing, responsive y estados visuales. Mantener la paleta Vena Digital:

- Azul noche: `#0A2142`
- Coral: `#FF6B5E`
- Amarillo: `#F5C935`
- Fondo blanco/marfil
